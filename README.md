# silicatPLC — Firmware дозаторного контроллера (ICP DAS 7188)

Встроенное ПО (firmware) для промышленного контроллера ICP DAS 7188XW на языке C. Управляет дозаторами на производстве силикатных изделий (кирпич, блоки). Реализует TCP-сервер с поддержкой Modbus TCP и собственного протокола.

## Назначение

Контроллер опрашивает 4 дозатора (известь + песок) через RS-485, обрабатывает данные и публикует их по Ethernet через Modbus TCP (порт 502) и собственный протокол (порт 10000). Данные сохраняются в EEPROM контроллера.

## Технические характеристики

| Параметр | Значение |
|---------|---------|
| Платформа | ICP DAS 7188XW (DOS-совместимый embedded PC) |
| Язык | C (ANSI C) |
| Протоколы | Modbus TCP (порт 502), custom TCP (порт 10000) |
| Интерфейс с устройствами | RS-485 / VCOM |
| Хранилище | EEPROM (параметры и калибровки) |
| Таймер цикла | аппаратный, ~1 сек |
| Сборка | Watcom C compiler (`7188xw.exe`) |

## Архитектура

```
silicatPLC/
├── dozator.C         # Основная логика: TCP-сервер, опрос дозаторов, CRC16, Modbus TCP
├── eeprom.c          # Чтение/запись параметров в EEPROM
├── var.h / vare.h    # Переменные состояния дозаторов (setPerf, curentPerf, statusDozator...)
├── TYPEEPR.H         # Структуры EEPROM
├── Tcpip32.h         # TCP/IP библиотека ICP DAS
├── VXCOMM.H          # VxComm (виртуальный COM) заголовки
├── tcpip32.lib       # Статическая TCP/IP библиотека
├── vcom3225.LIB      # VxComm библиотека
├── Dozator.mpj       # Проектный файл (Watcom Make)
└── autoexec.bat      # Автозапуск на контроллере
```

## Алгоритм работы

1. **Первый запуск** — опрос всех дозаторов, инициализация текущих/заданных производительностей
2. **Основной цикл** — `checkSystemNum()` поочерёдно опрашивает каждый дозатор по Modbus RTU через RS-485
3. **TCP-сервер** — принимает подключения на портах 10000 и 502 (Modbus TCP), отвечает на запросы SCADA
4. **CRC16** — все обмены защищены контрольной суммой (алгоритм CCITT CRC16)
5. **EEPROM** — уставки производительностей сохраняются при изменении

## Дозаторы

| Адрес | Тип | Назначение |
|-------|-----|-----------|
| `adrDoz[0]` | Известь 1 | curentPerfIzvest[0] / setPerfIzvest[0] |
| `adrDoz[1]` | Песок 1 | curentPerfSand[1] / setPerfSand[1] |
| `adrDoz[2]` | Известь 2 | curentPerfIzvest[2] / setPerfIzvest[2] |
| `adrDoz[3]` | Песок 2 | curentPerfSand[3] / setPerfSand[3] |

## Контекст

Разработано в 2014–2015 гг. для автоматизации силикатного завода. Работало в паре с SCADA-системой [silicat_scada](https://github.com/exclusivelogin/silicat_scada), которая визуализировала показания дозаторов и принимала команды оператора.
